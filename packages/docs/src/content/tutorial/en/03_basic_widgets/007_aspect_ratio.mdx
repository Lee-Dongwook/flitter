# AspectRatio Widget

The AspectRatio widget sizes its child to a specific aspect ratio, which is the ratio of width to height.

## Properties

- `aspectRatio`: number - The ratio of width to height
- `child`: Widget - The widget to size according to the aspect ratio

## Common Aspect Ratios

1. **16:9 (1.78:1)**
   - Widescreen video
   - Modern displays
   ```tsx
   AspectRatio({
     aspectRatio: 1.78,
     child: VideoPlayer()
   })
   ```

2. **4:3 (1.33:1)**
   - Traditional photos
   - Older displays
   ```tsx
   AspectRatio({
     aspectRatio: 1.33,
     child: Image()
   })
   ```

3. **1:1 (1:1)**
   - Square images
   - Profile pictures
   ```tsx
   AspectRatio({
     aspectRatio: 1,
     child: ProfileImage()
   })
   ```

## Use Cases

1. **Responsive Video Players**:
   ```tsx
   Container({
     width: "100%",
     child: AspectRatio({
       aspectRatio: 1.78,
       child: VideoPlayer()
     })
   })
   ```

2. **Image Galleries**:
   ```tsx
   GridView({
     children: images.map(image =>
       AspectRatio({
         aspectRatio: 1,
         child: Image(image)
       })
     )
   })
   ```

3. **Cards with Media**:
   ```tsx
   Column({
     children: [
       AspectRatio({
         aspectRatio: 1.78,
         child: HeaderImage()
       }),
       CardContent()
     ]
   })
   ```

## Best Practices

1. **Responsive Design**:
   - Use AspectRatio for consistent layouts
   - Consider different screen sizes
   - Test with various parent constraints

2. **Content Fitting**:
   - Use appropriate aspect ratios for content type
   - Consider content cropping
   - Handle overflow appropriately

3. **Performance**:
   - Avoid unnecessary nesting
   - Use const widgets when possible
   - Consider caching for lists

## Common Patterns
